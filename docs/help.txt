Cruce Game:
Famous card game, similar to whist.

CruceGame Usage:
cruceGame [-hv]
	-h, --help	Display help
	-v, --version Current Version of Cruce Game

No. of Players:
1-4

Goal:
Achieve the Game Points before your opponent.

Keyboard:
Standard keyboard to set other settings of game like player name, game points.
Left or Right arrow keys to choose the card. (a or d for card selection.)
Use ENTER to place the card on table. 
q to quit the game.
